1. Recurrence T(n)=T(n−1)+1T(n)=T(n-1)+1T(n)=T(n−1)+1​

We denote the function as T. 

Void Test(int n) {
    if(n>0){
        printf("%d",n)
        Test(n-1);
    }  
}

This has time complexity Θ(n)\Theta(n)Θ(n). Two primitive steps : comparison, printing. for each recurrence, this takes 2+T(n−1)2+T(n-1)2+T(n−1) steps. Two is a constant, so it is basically same as 1. n+1 function calls, n prints.

2. Recurrence T(n)=T(n−1)+nT(n) = T(n-1)+nT(n)=T(n−1)+n​

Void Test(int n){
    if(n>0){
        for(int i=0;i<n;i++) {
            printf("%d",n);
        }
        Test(n-1);
    }
}

Let's use substitution. Base Condition:

T(n)={1n=1T(n−1)+nn>1.\begin{equation*} T(n) = \begin{cases} 1 & \quad n=1 \\ T(n-1)+n & \quad n>1. \end{cases} \end{equation*}T(n)={1T(n−1)+n​n=1n>1.​​

Substitution: T(n)=T(n−1)+nT(n) = T(n-1)+nT(n)=T(n−1)+n =T(n−2)+(n−1)+n\quad\quad=T(n-2)+(n-1)+n=T(n−2)+(n−1)+n =T(n−3)+(n−2)+(n−1)+n\quad\quad=T(n-3)+(n-2)+(n-1)+n=T(n−3)+(n−2)+(n−1)+n ...\quad...... =T(n−k)+∑i=1ki\quad\quad=T(n-k)+\sum_{i=1}^k i=T(n−k)+∑i=1k​i =1+n(n−1)2\quad\quad=1+\frac{n(n-1)}{2}=1+2n(n−1)​

3. Recurrence T(n)=T(n−1)+log⁡nT(n)=T(n-1)+\log{n}T(n)=T(n−1)+logn​

Void Test(int n){
    if(n>0) {

So, in upper bond, log⁡n!<=log⁡nn\log{n!}<=\log{n^n}logn!<=lognn O(log⁡nn)=O(nlog⁡n)O(\log{n^n}) = O(n\log{n})O(lognn)=O(nlogn)
