# Turntable music player- roadmap

이 프로젝트는 태완이와 함께하는 Three.js + OpenGL + Blender + React + Django 크래시 코스의 첫번째 사이드 프로젝트이다. 

이 프로젝트는 각 기술들을 직접 사용해보는 것에 의의를 둔다. 

이 프로젝트는 음악을 재생하는 턴테이블을 3D로 보여주는, 시각적 재미가 있는 플레이어를 만든다.
