# 부타동

## 재료
간장, 설탕, 돼지고기, 밥, 파

## 레시피
1. 간장 2T에 물 약간, 설탕 1t 넣고 졸인다.  
2. 간장은 빼놓고, 돼지고기를 익힌다. 
3. 돼지고기가 익었으면 간장을 붓고 소스를 입힌디ㅏ. 
4. 밥 위에 돼지고기, 간장 붓고, 파로 가니쉬.
